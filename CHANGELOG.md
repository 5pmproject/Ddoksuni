# 변경 이력

## v1.1 (2026-01-27) - 사용자 친화적 입력 가이드 추가

### 개선사항
1. **ADL 점수 입력 개선**
   - 숫자 입력 → 슬라이더 UI로 변경
   - 실시간 점수 표시
   - 상세 평가 가이드 모달 추가
   - 간단 평가 방법 제공 (5가지 항목 체크)
   - 중증도별 상세 설명 (0-30점/40-60점/70-100점)

2. **건강보험 유형 설명**
   - 각 옵션에 상세 설명 추가
   - 도움말 버튼으로 추가 정보 제공
   - 직장가입자/지역가입자/의료급여 차이점 명확화
   - 확인 방법 안내

3. **장기요양등급 안내**
   - 각 등급별 상세 설명 추가 (1급-5급)
   - 점수 기준 표시
   - 신청 방법 3단계 안내
   - '아직 신청 안함' 옵션으로 명확화
   - 혜택 정보 제공

4. **발병일 입력 가이드**
   - 질환별 기준일 설명
   - 뇌졸중/암/골절 등 예시 제공
   - 중요성 설명 추가

### UI/UX 개선
- 물음표 아이콘 버튼 추가 (도움말 트리거)
- 인라인 힌트 텍스트 추가
- 컬러풀한 가이드 모달 디자인
- 그라데이션 배경으로 정보 계층 구분
- 아이콘 활용한 시각적 정보 전달

### 사용자 경험
- 의료 용어를 모르는 보호자도 쉽게 이해 가능
- 복잡한 평가를 간단한 체크리스트로 변환
- 각 항목의 중요성과 의미를 명확히 전달
- 실용적인 확인 방법 제공
