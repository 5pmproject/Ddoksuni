# 변경 이력

## v1.2 (2026-01-27) - Glasgow Coma Scale (GCS) 의식 상태 평가 도입

### 주요 변경사항
1. **의식수준 평가 방식 개선**
   - 단순 선택 방식 (명료/기면/혼미/혼수) → 표준화된 GCS 평가로 전환
   - 세 가지 영역 독립 평가:
     - 눈 뜨기 반응 (Eye Opening): 1-4점
     - 언어 반응 (Verbal Response): 1-5점
     - 운동 반응 (Motor Response): 1-6점
   - 총점 3-15점으로 객관적 의식 수준 평가

2. **실시간 GCS 점수 계산**
   - 세 영역 선택 시 자동으로 총점 계산
   - 중증도별 색상 코드:
     - 15점: 정상 (초록)
     - 13-14점: 경미한 장애 (파랑)
     - 9-12점: 중등도 장애 (노랑)
     - ≤8점: 중증 장애 (빨강)

3. **상세 GCS 평가 가이드**
   - 각 영역별 점수 기준 상세 설명
   - 실제 환자 상태 예시 제공
   - 지남력, 주의력, 인지기능 설명
   - 평가 팁 및 주의사항
   - 의료진 확인 방법 안내

4. **데이터베이스 스키마 업데이트**
   - GCS 점수 저장 필드 추가 (gcs_total, gcs_eye, gcs_verbal, gcs_motor)
   - 기존 환자 데이터 자동 변환
   - GCS 검색 인덱스 추가

### 의학적 정확성 향상
- 국제 표준 의식 평가 도구 적용
- 의료진과의 커뮤니케이션 용이
- 재활 계획 수립에 더 정확한 데이터 제공
- 섬망(Delirium), 혼란(Confusion) 상태 구분 가능

### 사용자 경험
- 진료기록지의 GCS 점수 그대로 입력 가능
- 각 항목별 명확한 설명으로 정확한 평가 가능
- 의료진에게 문의할 내용 명확화

## v1.1 (2026-01-27) - 사용자 친화적 입력 가이드 추가

### 개선사항
1. **ADL 점수 입력 개선**
   - 숫자 입력 → 슬라이더 UI로 변경
   - 실시간 점수 표시
   - 상세 평가 가이드 모달 추가
   - 간단 평가 방법 제공 (5가지 항목 체크)
   - 중증도별 상세 설명 (0-30점/40-60점/70-100점)

2. **건강보험 유형 설명**
   - 각 옵션에 상세 설명 추가
   - 도움말 버튼으로 추가 정보 제공
   - 직장가입자/지역가입자/의료급여 차이점 명확화
   - 확인 방법 안내

3. **장기요양등급 안내**
   - 각 등급별 상세 설명 추가 (1급-5급)
   - 점수 기준 표시
   - 신청 방법 3단계 안내
   - '아직 신청 안함' 옵션으로 명확화
   - 혜택 정보 제공

4. **발병일 입력 가이드**
   - 질환별 기준일 설명
   - 뇌졸중/암/골절 등 예시 제공
   - 중요성 설명 추가

### UI/UX 개선
- 물음표 아이콘 버튼 추가 (도움말 트리거)
- 인라인 힌트 텍스트 추가
- 컬러풀한 가이드 모달 디자인
- 그라데이션 배경으로 정보 계층 구분
- 아이콘 활용한 시각적 정보 전달

### 사용자 경험
- 의료 용어를 모르는 보호자도 쉽게 이해 가능
- 복잡한 평가를 간단한 체크리스트로 변환
- 각 항목의 중요성과 의미를 명확히 전달
- 실용적인 확인 방법 제공
